项目实现：1.定义remote_host类，类里面的command，put，get，run函数   2.创建服务器的配置文件database  3.定义程序入口

项目执行流程：程序启动-选择分组和服务器-输入linux命令-启动多线程远程连接多个服务器-远程服务器返回命令执行结果-打印返回结果
